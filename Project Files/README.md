# 🚖 Ucab - Backend

The backend of **Ucab** is built using **Node.js**, **Express.js**, and **MongoDB**. It provides secure REST APIs for authentication, ride booking, payment handling, user management, and real-time ride updates.

---

## 📖 Overview

Ucab is a MERN stack cab booking application that connects passengers with nearby drivers through a reliable backend service. The backend manages authentication, ride requests, fare calculation, database operations, booking history, and communication between users and drivers.

---

- User Authentication

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or above)
- npm (v8 or above)
- MongoDB

### Installation

```bash
git clone <repository-url>

cd backend

npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=

DB_CONNECT=

JWT_SECRET=

GOOGLE_MAPS_API=
```

*(Modify the variables according to your project configuration.)*

---

## ▶️ Run Server

```bash
npm start
```

or

```bash
node server.js
```
